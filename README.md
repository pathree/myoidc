# oidc

这个项目是学习和研究OpenID Connect协议，使用NodeJS实现一个OAuth2.0的Client。

## 运行方法

1. 服务器运行 node index.js 监听80端口
2. 客户端配置www.example.com指向服务器地址
3. 浏览器打开http://www.example.com/
